Preguntas frecuentes

¿Tienes dudas?

¿Necesito saber programar para usar este proyecto?
No. El proyecto está construido únicamente con HTML y CSS, que no son lenguajes de programación. La guía explica cada parte desde cero, pensada específicamente para principiantes. Solo necesitas instalar Visual Studio Code y Live Server, y la guía te lleva paso a paso.
¿Puedo usar este proyecto para mi tarea universitaria?
Sí, con un ajuste importante: el proyecto es de referencia académica. Debes personalizarlo — cambiar los textos, colores, imágenes y temática — para adaptarlo a tu asignación específica. La guía incluye un capítulo dedicado a cómo hacer esas adaptaciones en menos de una hora.
¿Qué recibo exactamente al comprar?
Recibes dos archivos descargables: (1) La guía en formato .pdf con 12 capítulos y código real del proyecto, y (2) el proyecto completo comprimido en .zip, listo para descomprimir, abrir y adaptar según lo requieras en VS Code. Todo de forma inmediata al completar la compra.
¿El proyecto funciona solo con HTML y CSS, sin JavaScript?
Sí, el 100% del proyecto funciona sin una sola línea de JavaScript. Los efectos hover del menú, las animaciones de entrada, el zoom de imágenes, la validación del formulario y hasta el mensaje de confirmación al enviar — todo se logra con CSS puro. Ideal para asignaciones que solo piden HTML y CSS.
¿Puedo adaptar el proyecto a una temática diferente a biblioteca?
Claro que SÍ. La guía incluye 7 ideas concretas de adaptación: turismo, portafolio personal, etc. La estructura de carpetas, el CSS y el diseño responsive funcionan exactamente igual con cualquier temática. Solo cambias y adaptas textos, colores e imágenes.
